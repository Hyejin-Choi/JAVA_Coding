## JAVA_Coding

## 최종 기말 프로젝트
### **주제명**  : 군인 관리 프로그램

**프로그램 설계**
프로젝트를 수행하면서 프로그램을 구성하기 위해서 처음에는 프로그램의 내용을 어떻게 구성할 것인지 집중적으로 생각하였습니다. 프로그램을 구성들에 대한 설명은 다음과 같습니다.

1.아이디와 비밀번호 설정

2.군인의 직업과 이름, 입대날짜를 설정

3.파일에 저장

4.군인목록을 검색하면 검색된 군인의 이름, 직업,계급,월급,제대일 출력 

5. 재검색을 하거나, 끝내기 기능
